## 서론

정보처리기사는 비전공자인 내가 IT 업계에서 밥 벌어 먹기 위한 시민권과 같다.

이것을 따지 않고 업계에서 살아가는건 좀 과장보태서 불법체류자와 같다.

물론 정보처리기사가 없다고 취업을 못하는건 아니다.

하지만 필기까지 딴 마당에 실기를 안한다는건

나 이철욱이라는 사람에게는 용납될 수 없다.

조져버리겠다 정처기.

나의 매서운 정답에 오들오들 떨 준비나 하거라.

기열.
