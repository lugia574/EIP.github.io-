# D. 프로그래밍 언어 활용

## 예외처리

    예외가 발생했을 때 해당 문제에 대비해 작성해 놓은 루틴을 수행하도록 하는것

```py
try:
    r=int(input('실행할 코드임 정수를 입력: ')) # 숫자 입력
    pi=3.141592
    print('원의 반지름: ', r)
    print('원의 둘레: ', 2*pi*r)
    print('원의 면적: ', r*r*pi)

except:
    print('예외발생 정수입력하셈')
else:
    print('예외발생 안되면 이걸 출력')
finally:
    print('무조건 실행할 코드 마무리~')
```
